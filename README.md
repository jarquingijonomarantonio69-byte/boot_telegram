# Bot De Telegram

Un bot de telegram donde todos los mensajes que reciba se mostraran en mi consola usando la tecnologia Long Polling

Donde en el archivo "telegramboot\src\main\java\com\sacavix\telegramboot\service\TelegramBoot.java"
esta el nombre del bot y la key de este, los cuales permiten la conexion con el bot.
En este mismo archivo se puede modificar las respuestas predeterminadas dependiendo la necesidad del uso del bot

Entrando al chat de mi bot t.me/Jax042_bot y ejecutando el proyecto se podra ver en consola los mensajes que le escriban en el chat
